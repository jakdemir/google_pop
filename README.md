# Google POP3 Client

A Java-based POP3 client for connecting to Google Mail servers.

## Overview

This project implements a POP3 (Post Office Protocol version 3) client in Java, specifically designed to work with Google's email servers.

## Features

- Connect to Gmail via POP3 protocol
- Retrieve and manage emails programmatically

## Technologies

- Java
- POP3 Protocol

## Project Structure

```
src/          # Java source files
lib/          # External libraries
```

## Setup

1. Import as a Java project in your IDE
2. Configure your Gmail account settings
3. Enable "Less secure app access" or use App Passwords
